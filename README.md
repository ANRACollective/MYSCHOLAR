# MyScholar

**Malaysia's free platform to plan a career, find a scholarship, and land an internship.**

Live at [myscholar.my](https://myscholar.my)

---

## The problem

Most Malaysian students figure out their future with scattered Google searches, outdated PDFs, and word of mouth. Scholarship deadlines get missed. Good internships stay hidden. Nobody tells you which careers are actually worth studying for, or how to get there from SPM.

The information exists. It is just buried, fragmented, and hard to trust.

## What MyScholar does

One place, three stages of a student's journey:

**Plan — MyJourney**
200+ careers with real Malaysian pathways: which SPM subjects you need, which pre-U route to take, which local universities offer it, salary ranges, and demand outlook.

**Fund — MyScholar**
200+ scholarships and study loans, filterable by field, level, citizenship, and deadline. JPA, MARA, GLCs, banks, foundations, and foreign governments in one searchable directory.

**Launch — MyIntern**
Curated internships with verified links, stipend info, and deadline tracking, plus a CV builder and interview prep.

Everything is free, mobile-first, and built for the 80%+ of students who browse on their phones.

## Why it works

- **Verified, not scraped.** Listings are checked and corrected, so students can trust what they see.
- **Deadline-aware.** Opportunities closing soon are surfaced first.
- **No login wall.** Bookmark and explore instantly, no account required.

## Built with

A deliberately lean stack: single-file HTML apps, served on GitHub Pages, backed by Supabase, with automated link-health checks keeping listings current. No framework, no build step, no server to maintain.

| | |
|---|---|
| Frontend | Vanilla HTML / CSS / JS (single-file SPAs) |
| Data | Supabase (Postgres) |
| Hosting | GitHub Pages |
| Email | Resend |
| Analytics | GA4 |

## The story

MyScholar is built and maintained by one person, with no formal coding background, using AI as a co-developer. It started as a single scholarship list and grew into a three-product ecosystem covering 400+ verified opportunities.

It is still early days and I am adding more regularly, but it is already helping real students.
