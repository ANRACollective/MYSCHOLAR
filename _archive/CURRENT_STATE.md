## What the app is

MyScholar.my is a free, no-login scholarship and internship finder for Malaysian students. It is a solo-built, static web app hosted on GitHub Pages with a Supabase database backend. There is no backend server — all logic runs client-side in two single-file HTML SPAs.

## Features that exist

### index.html (Scholarships SPA)
- Supabase data fetch via `scholarships_full` view + `initData()`
- Search by keyword
- Filters: category, coverage, destination, eligibility, grade, deadline month
- Deadline display with 60-day urgency window
- Bookmarks via localStorage (no login required)
- Correction/feedback submission form
- Scholarship quiz (eligibility matcher)
- GA4 events: `search_query`, `filter_applied`, `correction_submitted`, `quiz_completed`
- Light/ice aesthetic — fixed design language

### internships.html (Internships SPA)
- Supabase data fetch from `internships` table
- Company cards with logos (`logo_url` / `logo_domain`)
- Search and filter (industry, study level, stipend)
- Apply links out to external URLs
- Dark glass aesthetic — visually disconnected from index.html branding/header
- GA4 events partially implemented

### SEO layer
- 74 static HTML landing pages in `SEO/` folder
- 66 scholarship-category pages (`biasiswa-*.html`)
- 4 internship pages, how-to guides, comparison pages
- Two sitemaps: `sitemap.xml` (root) + `SEO/sitemap-seo.xml`

## Known issues / mess areas

1. **No git workflow** — no `.git` folder in `MYSCHOLAR/`. Every update is a manual file upload via GitHub web UI. This is slow, error-prone, and historically caused a byte corruption incident that broke `initData()` silently.

2. **Internship URL quality** — 19 records have `unverified` or `suspected` status. 3 of these are currently shown to users. The URLs likely point to generic career pages rather than specific application portals.

3. **internships.html is visually orphaned** — dark glass design shares no header, logo, or nav with index.html. A student navigating from the scholarship page to internships feels like they've left the site.

4. **Leftover clutter in repo** — `SEO/test.html`, empty `CONTEXT/`, `FAVICON/`, `_test_move/` directories, and a likely duplicate `_assets/myscholar_profile_photo (1).svg` all exist but serve no purpose.

5. **OG image is oversized** — `_assets/myscholar-og.png` is 293 KB. The `.jpg` version at 42 KB should be preferred.
