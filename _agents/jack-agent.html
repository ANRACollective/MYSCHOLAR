GitHub PAT Setup
You need a Personal Access Token so JACK can deploy pages to ANRACollective/MYSCHOLAR. Takes 3 minutes.
1
Go to github.com/settings/tokens/new (you must be logged in as the repo owner).
2
Set Note to JACK-SEO-Agent. Set Expiration to No expiration (or 1 year).
3
Under Select scopes, tick only repo (full control of private repositories). Nothing else needed.
4
Click Generate token. Copy the token — it starts with ghp_. GitHub only shows it once.
5
Paste it into the GitHub PAT field in JACK's sidebar. It's stored only in your browser session — never sent anywhere except GitHub's API.
